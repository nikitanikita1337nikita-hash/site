# VK: Приватность профиля

ВКонтакте по умолчанию открыт. Чтобы скрыть свою жизнь от посторонних, переведите профиль в **Закрытый статус**.
::: tip Как найти эти настройки
Нажмите на свою аватарку ➔ **Настройки** ➔ **Приватность**
:::
![VK Privacy](/img/vk-privacy.png)
*Рис. 3. Настройки конфиденциальности VK*

**Рекомендуемые параметры:**
* Режим: Закрытый профиль
* Скрытые друзья: Только я
