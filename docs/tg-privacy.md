# Telegram: Приватность

Telegram позволяет скрыть номер телефона даже от тех, кто сохранил ваш контакт. Это критически важно для защиты от деанонимизации.
::: tip  Как найти эти настройки
**Настройки** ➔ **Конфиденциальность**
:::

![TG Privacy Settings](/img/tg-privacy.png)
*Рис. 1. Настройки конфиденциальности Telegram*
::: info Рекомендуемые настройки
Установите значение «Никто» для Номера телефона, Времени захода и Пересылки сообщений.
:::
