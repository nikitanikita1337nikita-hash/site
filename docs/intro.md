#  База безопасности

Прежде чем настраивать соцсети, нужно понять три столпа цифровой гигиены.

### Пароли
Минимум 12 символов, отсутствие личных данных в строке.

### 2FA
Второй фактор защиты — это стена, которую нельзя обойти.

### Приватность
Меньше информации в открытом доступе — меньше шансов для социнженерии.
