# VK ID: Безопасность

Центр управления вашими данными в ВК — это VK ID. Здесь настраивается второй фактор защиты.
::: tip  Как найти эти настройки
Настройки ➔ **Управление аккаунтом (VK ID)** ➔ **Безопасность и вход**
:::
