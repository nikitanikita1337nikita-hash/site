# Telegram: Защита входа

Взлом через перехват СМС — самая частая угроза. Её решает **Облачный пароль**.
::: tip  Как найти эти настройки
**Настройки** ➔ **Конфиденциальность** ➔  **Безопасность**
:::

![Telegram Security](/img/tg-security.png)
*Рис. 2. Настройки безопасности Telegram*
::: warning Важно: Активные сеансы
Если на скриншоте вы видите число сеансов больше 1, убедитесь, что все устройства принадлежат вам.
:::
