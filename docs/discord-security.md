# Discord: Угон аккаунта

Discord часто взламывают через поддельные QR-коды. Аутентификатор — единственное спасение.
::: tip Как найти эти настройки
Нажмите на **шестерёнку (Настройки пользователя)** в левом нижнем углу:
* Для включения 2FA: вкладка **Моя учётная запись**.
* Для проверки сторонних ботов: вкладка **Авторизованные приложения**.
:::
